# Basictofu Tap

## How do I install these formulae?

`brew install basictofu/tap/<formula>`

Or `brew tap basictofu/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "basictofu/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
